Project: Orienteering in Mendon Ponds Park
Author: Jesse Vogel
Date Created: September 13th, 2016

Description: Give the program a file containing necessary event details and the program will generate a map with
 the best path drawn in red, (named map.png.) It will also generate a text file that describes the path in plain
 english, (named directions.txt.)

To run the program:
1. place program.py, terrain.png, and mpp.txt into an empty directory.
2. In the same directory, place your input file to pass to the program.
3. run the command 'python program.py inputfile.txt'
4. view results in map.png and directions.txt
